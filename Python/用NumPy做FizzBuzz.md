写一个程序，输出从 1 到 n 数字的字符串表示。

1. 如果 n 是3的倍数，输出"Fizz"；
2. 如果 n 是5的倍数，输出"Buzz"；
3. 如果 n 同时是3和5的倍数，输出 "FizzBuzz"；
4. 否则输出　n.


```python
import numpy as np
x = np.arange(1, 101)

np.select([(x%3 == 0) & (x%5 == 0), (x%3 == 0), (x%5 == 0)], ['FizzBuzz','Fizz','Buzz'], x)
```
