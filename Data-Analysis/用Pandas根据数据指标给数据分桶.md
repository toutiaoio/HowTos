定义用户级别及范围


```python
bins = [0, 500, 4000, 10000, 50000, 150000, 800000, 1500000]
labels = ['V0', 'V1', 'V2', 'V3', 'V4', 'V5', 'V6']
```

用 Pandas 的 `cut` 方法给 df 增加 `level` 字段表示用户级别， `right=False` 表示左闭右开区间


```python
df['level'] = pd.cut(df['growth_point'], bins, labels=labels, right=False)
```
